# Генератор цитат

**Автор:** Владислав Морозов

## Описание программы

Генератор цитат — это desktop-приложение на Python с графическим интерфейсом (Tkinter), которое позволяет:

- Генерировать случайные цитаты из предустановленной коллекции
- Добавлять новые цитаты с проверкой корректности ввода
- Просматривать историю сгенерированных цитат с указанием времени генерации
- Фильтровать историю по автору и теме
- Автоматически сохранять историю и базу цитат в JSON-файлы

## Предустановленные цитаты

**Темы:** Мораль, Мудрость, Жизнь, Счастье, Знания, Мотивация

## Технические требования

- Python 3.6 или выше
- Библиотека Tkinter (обычно входит в стандартную поставку Python)
- Операционные системы: Windows, macOS, Linux
